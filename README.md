# GreenAura 🌱 - E-commerce de Bienestar y Naturaleza

Una plataforma web inmersiva y elegante diseñada para presentar productos naturales de calidad premium. Este proyecto combina un diseño UI/UX limpio y orgánico con la integración de videos cinemáticos generados por IA para mostrar el proceso artesanal detrás de cada producto.

## 📖 Descripción del Proyecto

GreenAura busca transmitir bienestar, pureza y conexión con la naturaleza a través de su interfaz. La plataforma exhibe un catálogo curado de productos naturales, donde cada tarjeta de producto está diseñada para alojar micro-videos inmersivos que ilustran su origen y proceso de elaboración.

**Productos Destacados en la Interfaz:**
* **Cúrcuma Raíz Oro:** Antiinflamatorio milenario de maceración lenta.
* **Aceite de Lavanda Montañesa:** El suspiro de los Alpes para una relajación profunda.
* **Miel de Selva Virgen:** Energía natural e inmunidad con texturas doradas.
* **Extracto de Té Verde Ancestral:** Antioxidante y vitalidad.

## ✨ Características Principales (Features)

* **Diseño Limpio y Minimalista:** Interfaz enfocada en la claridad, utilizando una paleta de colores coherente (verdes, dorados, tonos tierra y blancos suaves).
* **Multimedia Integrada:** Soporte para videos inmersivos dentro de las tarjetas de presentación, con transiciones suaves y fluidas.
* **Identidad Visual Coherente:** Tipografía y maquetación estructuradas para transmitir una sensación "premium" y orgánica.
* **Responsive Design:** Adaptabilidad a diferentes dispositivos (Mobile, Tablet, Desktop) manteniendo la experiencia sensorial.
